
# jovanops.github.io

Personal portfolio website for JovanOps.

Live website:

https://jovanops.github.io

This portfolio is not only a static HTML page.  
It includes a custom domain, Cloudflare DNS/SSL, GitHub Pages deployment, sitemap.xml, robots.txt, favicon setup, Google Search Console indexing, and multiple project pages.
